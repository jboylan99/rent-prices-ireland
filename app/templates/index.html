Home

Rent Prices Ireland is an application designed to allow students to view rent prices around Ireland. If you're a student looking for a place to stay in college, this site should have all the information you need. You can view information by county in County Statistics and you can see the countrywide breakdown in National Statistics.

All data was based on a publicly available dataset from the Central Statistics Office at CSO.ie.


Predictive Models Used

Many different machine learning algorithms were used in order to make the findings as accurate as possible. The following algorithms were used to predict rent prices:


  • Moving average

  • Linear regression

  • K-nearest neighbour

  • Gradient boosting

  • Long short term memory

  • Random forest

  • ARIMA

After extensive testing, ARIMA was chosen as the algorithm to be used.


What is ARIMA?

Autoregressive integrated moving average (ARIMA) is a machine learning algorithm used to solve time series problems. Time series analysis is the study of a series of data points ordered by time, and ARIMA studies this data by examining its previous values to forecast future results.
